# Relatórios de Atendimento Tomasoni

Aplicação corporativa para cadastro de máquinas, técnicos e registros de atendimento.

## Stack

- Next.js
- Supabase
- Vercel

## Configuração local

1. Copie `.env.example` para `.env.local`.
2. Preencha `NEXT_PUBLIC_SUPABASE_ANON_KEY` com a chave pública do projeto Supabase.
3. Instale dependências com `npm install`.
4. Rode `npm run dev`.

## Supabase

O schema inicial está em `supabase/migrations/001_initial_schema.sql`.
